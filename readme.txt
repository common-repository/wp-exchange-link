wp-exchange.link is WordPress Pingback Link Exchange System that you can get free permanent backlinks and traffic to your WordPress site.

== Description ==

 When WP Exchange Link Plugin is activated, during saving post data to the database, a list of pingback links of plugin members' sites is added at the bottom of the post automatically. Pingback is the link to other WordPress websites from your post. If pingback link is sent to a WordPress site, the site will send a link back from their comment section. With WP Exchange Link plugin, everything is automatically done to get free traffic. Pingback links to other plugin members are added at the bottom of posts. Pingbacked members sites will return backlinks to your site from their comment section.

 When you visit the site of wp-exchange.link, you can check points you have and charge points.

 This plugin uses API of wp-exchange.link to get information of members to build pingback lists.

== Installation ==

1. Download the plugin file "wp-exchange-link.zip" from [http://wp-exchange.link/downloads](http://wp-exchange.link/downloads).
1. WordPress Administration Dashboard > Plungins menu > Add New > Upload Plugin
1. Select the downloaded file "wp-exchange-link.zip".
1. Click "Install Now".
1. Click "Activate Plugin".
1. Use the WP Exchange Link Settings menu to configure the plugin. The input information must be identical to the information set in wp-exchange.link.
